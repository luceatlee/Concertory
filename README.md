# 🎤 CONCERTORY

> K-POP 콘서트 무대 아카이빙 서비스

팬들이 직접 기록하고 탐구하는 K-POP 콘서트 무대 아카이브입니다.  
세트리스트, 무대 영상, 의상, 한 줄 후기까지 — 그 날의 무대를 영원히.

<br />

## 🛠 Tech Stack

| Category | Stack |
|---|---|
| Framework | Next.js 15 + TypeScript |
| Styling | Tailwind CSS |
| Database | Supabase |
| Deployment | Vercel |

<br />

## 🚀 Getting Started
```bash
git clone https://github.com/luceatlee/Concertory.git
cd concertory
npm install
npm run dev
```
